# 宜诺TCP服务技术分享

### Configure it

Modify the DB configuration in `/src/main/resources/jdbc.properties`

### Deploy it

Package it, `mvn clean package -DskipTests`

### Run it
```
$ java -Xms128m -Xmx128m -jar target/socket-0.0.1-SNAPSHOT.jar
```

### Tests

使用Telnet来访问服务器:

```
$ telnet localhost 5678
```
